Lesson 6: HTML Tables

Topic Overview

HTML Tables are used to organize data into rows and columns. Think of them like spreadsheets where you can clearly show information like timetables, score sheets, or student data. Tables are created using the <table> element, and inside it, we use <tr> for table rows, <td> for data cells, and <th> for header cells.

Understanding how to structure a table in HTML helps you present complex data in an organized and easy-to-read manner. It's very useful for web apps, dashboards, and information-heavy pages.

Code Example

<table border="1" cellpadding="10"> <tr> <th>Name</th> <th>Age</th> <th>Class</th> </tr> <tr> <td>John</td> <td>12</td> <td>JSS 1</td> </tr> <tr> <td>Mary</td> <td>13</td> <td>JSS 2</td> </tr> </table>
