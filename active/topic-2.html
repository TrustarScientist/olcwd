🧱 Topic 2: Understanding HTML Tags and Structure

📖 Topic Overview

Imagine building a house. You need bricks, doors, windows, and a roof, right? Well, websites are like houses too! And instead of bricks, we use something called HTML tags.

HTML stands for HyperText Markup Language. It’s what we use to tell the computer: “Hey! This is a heading”, or “That’s a picture!”, or “Please show a paragraph here!”

HTML is made up of many small building blocks called tags. These tags tell the browser how to show each part of your webpage.

💡 Tags come in pairs!

Most tags have an opening tag and a closing tag. Example:

<p>This is a paragraph</p>
  • <p> is the opening tag
  • </p> is the closing tag

📦 Common Tags You’ll Use a Lot:

  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <img> – Image
  • <a> – Link
  • <ul> and <li> – List

When you combine many tags, you build a full webpage – just like bricks forming a house!

🛠️ Editable Code Example

<!DOCTYPE html>
<html>
  <head>
    <title>HTML Tags Example</title>
  </head>
  <body>
    <h1>My First Page</h1>
    <p>Hello! I love learning HTML.</p>
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </body>
</html>

Try changing some text or adding another list item!
